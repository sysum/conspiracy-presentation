# The Hathaway Dossier

A conspiracy-style Marp presentation examining the historical evidence around Shakespeare's marriage to Anne Hathaway — the name discrepancy in the 1582 marriage records, the "second-best bed" bequest, and what scholarly consensus actually says vs. what the gaps in the archive invite people to imagine.
